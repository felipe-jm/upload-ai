- [shadncn/ui](https://ui.shadcn.com/)

## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/felipe-jm/upload-ai-web

# Go into the repository
$ cd upload-ai-web

# Install dependencies
$ pnpm i

# Run the app
$ pnpm run dev
```
